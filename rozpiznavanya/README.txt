Цей проект може працювати без підключення до інтернету 
Проте перший запуск програми повинен бути обов'зково з ПІДКЛЮЧЕННЯМ ДО ІНТЕРНЕТУ для завантаження усіх бібліотек

Для цього потрібно завантажити такі бібліотеки
Просто вставте цю команду в термінал:

pip install vosk stanza pydub pyqt5 pillow noisereduce rapidfuzz 

Перед першим запуском зайдіть у файл work_NLP.py та виконайте певні дії:
    1. У файлі work_NLP.py знайдіть рядки 11 та 12 та роздокументуйте їх
    2. Задокументуйте код на 15 рядку

    Далі запуск програми:
        1. Увійдіть у папку rozpiznavanya за допомогою cd 
        2. Запустіть програму за допомогою команди      python speech_recognition.py
        3. Дочекайтеся поки всі залежності встановляться та завантажиться програма(тобто коли ви зможете взаємодіяти з інтерфейсом)
    
    Наступний крок можна виконати якщо ви хочете щоб програма працювала повністю офлайн(тобто без підключення до інтернету):
        1. Зайдіть у файл work_NLP.py знайдіть рядки 11 та 12 та задокументуйте їх
        2. Рядок 15 - роздокументуйте
        3. Збережіть файл
        Тепер програма може працювати без підключення до інтернету

    Запуск здійснюйте за допомогою команди      python speech_recognition.py
    У папці rozpiznavanya
